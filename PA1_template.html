Imputing missing values

Note that there are a number of days/intervals where there are missing values (coded as NA). The presence of missing days may introduce bias into some calculations or summaries of the data.

1. Calculate and report the total number of missing values in the dataset

na.indices <- which(is.na(data$steps))
not.complete.cases <- length(na.indices)

All the missing values are in the steps column. The total number of missing values is 2304 steps, i.e. there are 2304 rows in the data frame where we do not know the number of steps that corresponding to such intervals.

2. Devise a strategy for filling in all of the missing values in the dataset.

I decided to put in each missing values the value that appears for the same interval in the activity daily pattern. That is, I extract the values from the average of all the 5 minute intervals across all the days.

impute.missing.value <- function(x) {
    aggregated.daily.pattern$steps[aggregated.daily.pattern$interval == data$interval[x]]
}

3. Create a new dataset that is equal to the original dataset but with the missing data filled in.

data.with.imputed.missing.values <- data
imputed <- sapply(na.indices, impute.missing.value)
data.with.imputed.missing.values$steps[na.indices] <- imputed
